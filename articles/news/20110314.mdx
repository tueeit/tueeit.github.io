---
title: 工會籌組討論會會議記錄
createdAt: '2011-03-14'
---

科學園區勞動人權協會在三月 12、13 日兩天先後在善化、台北、中壢、竹北召開了工會籌組討論會，延續並確認第一次竹北場討論的內容，主要包括：

```
一、確認工會名稱為「電子電機資訊產業工會」。
二、初步認為會址應設於新竹或台北地區。
三、與會人員交換了各廠工作量與延後下班的概況，以及同事關係、主管與下屬的關係。
四、工會應辦理勞工教育，與會員談談勞動法規的重要內容。
```

另外，會後與會成員要持續留意的事項如下：

```
一、在發起人會議召開之前，請大家持續私底下找可信賴的成員，或在PTT以外的管道也做招募會員的宣傳。
二、請大家思考有關工會的重要訴求（盡量涵蓋工程師、作業員關心的事項），以及會址設立選擇的優缺點。
三、在建立網路討論平台之前，有任何意見暫時先寄至 kwktw236@gmail.com。
四、發起人會議，評估視訊會議的可能性。
```

園區勞權會並且報告了幾個事項：

```
一、責任制合法性的爭議，及勞基法84-1的內容及問題。
二、會製作發起人簽署表，請大家正式簽署發起電資工會（評估網路簽署的可行性）。
三、預定近日召開發起人會議，如確定當日無法出席，可先不簽發起人，會後直接簽入會申請書。
```
